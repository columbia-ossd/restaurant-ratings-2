# Restaurant Ratings
This program allows users to see ratings for restaurants based on a specific cuisine.

To run the program, run:
```
java RatingAnalyzer
```

# Configuration
The file "reviews.csv" contains the listing of restaurants and their info.

"Review.java" represents a single restaurant review.

"ReviewReader.java" contains code to read from the input file.

"RatingAnalyzer.java" contains the code to interact with the user.
